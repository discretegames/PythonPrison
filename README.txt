An entry for the PyWeek31 week-long game jam contest https://pyweek.org/31/

Theme: "Cops"

Idea: You are in a prison and have to build and execute code to bribe your way out.

Game title: Python Prison

Made by: @discretegames

Usage:

pip install -r requirements.txt
python run_game.py


Controls:

WASD/Arrows = move
Shift = sprint
Space = pull
12 = set execution region
34 = set output region
R = run execution region
Esc = reset level
M = toggle mute
I = toggle info
